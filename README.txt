Aplikácia Ľudotvor

Použitie:

-------------------------------------------------------------------------------
a) lokálne z repozitára
-------------------------------------------------------------------------------
V príkazovom riadku použite tieto príkazy:

git clone https://github.com/MarianSulgan/ludotvor.git
cd ludotvor
npm run install
npm start

Poznámka: Musíte mať nainštalovené npm. Stiahnuť si ho môžete napríklad na: https://nodejs.org/en/.

-------------------------------------------------------------------------------
a) lokálne z priložených zdrojových súborov
-------------------------------------------------------------------------------
V príkazovom riadku použite tieto príkazy:

cd ludotvor
npm run install
npm start

Poznámka: Musíte mať nainštalovené npm. Stiahnuť si ho môžete napríklad na: https://nodejs.org/en/.

-------------------------------------------------------------------------------
a) na webe, online
-------------------------------------------------------------------------------
Aplikácia funguje aj online, na adrese:

http://ludotvor.herokuapp.com
